# laboutique-react

# Description

Une SPA d'une boutique pour afficher des produits et des catégories.

# Installation

Entrer la commande "npm install" dans un terminal à la racine du projet.

# Utilisation

Ouvrir un terminal à la racine du projet et entrer "npm run dev".
